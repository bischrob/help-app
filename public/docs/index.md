Thank you for using CatMapper. This guide will help you get started with the CatMapper application, although the user guide is still a work in progress. Please feel free to contact us at [support@catmapper.org](mailto:support@catmapper.org) for direct support.

You can learn more about CatMapper in our white paper[^1] available at this [link](https://osf.io/preprints/socarxiv/n6rty/).

[^1]: Hruschka, Daniel J., Robert Bischoff, Matt Peeples, Sharon Hsiao, and Mohamed Sarwat (2022) CatMapper: A user-friendly tool for integrating data across complex categories. SocArXiv Papers. https://osf.io/preprints/socarxiv/n6rty/
